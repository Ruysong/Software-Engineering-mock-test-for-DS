# 소프트웨어공학 지문형 모의고사

소프트웨어공학 기말고사 대비용 지문형 객관식 모의고사입니다.

## 구성

- 18개 강화 지문
- 지문별 객관식 5문항, 총 90문항
- 문제 일괄 채점 및 즉시 해설
- 오답 자동 저장소
- 별표 문제 저장소
- 브라우저 `localStorage` 기반 학습 기록 저장

## 데이터 갱신

저장소의 `소프트웨어공학_지문강화형_문제통합_최종본.md`를 수정한 뒤 아래 명령으로 문제 데이터를 다시 생성합니다.

```bash
node build-data.mjs
```

## 배포

의존성 없는 정적 웹사이트이며 Vercel에서 별도 빌드 설정 없이 배포할 수 있습니다.
