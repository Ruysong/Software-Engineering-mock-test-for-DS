# C언어 기말 개념예제 문제은행 · 플래시카드

시스템프로그래밍 기말(9~14주차) 대비 **인터랙티브 문제은행**. 좌우 분할 플래시카드로 한 문제씩 풀고, 버튼/단축키로 정답을 펼치며, 자기평가(🔴🟡🟢⭐)를 저장해 복습할 수 있습니다.

원본 코드 기반: [Ruysong/C_language_basic](https://github.com/Ruysong/C_language_basic)

## 구성
- 총 **151문제** (9주 25 · 10주 34 · 11주 30 · 12주 21 · 13주 29 · 14주 12)
- 유형: 빈칸 / 오류수정 / 실행추적 / 변형구현 / 개념
- 모든 문제에 정답 · 해설 · 함정(주의) 포함

## 사용법
- **정답 보기**: 카드 오른쪽 클릭 / `Space` / `↓`
- **다음·이전**: `→`(또는 `Enter`) / `←`  ※ 정답 안 본 상태에서 `→`는 먼저 정답을 펼침
- **자기평가**: `1` 🔴모름 · `2` 🟡애매 · `3` 🟢앎 · `4` ⭐별표 (같은 키 다시 누르면 해제)
- **검색**: `f`, **필터 닫기**: `Esc`
- 상단 **필터**(주차·유형·난이도·표시·검색)는 ⚙ 버튼으로 열고 닫기
- **셔플**로 순서 섞기

자기평가 표시는 브라우저 `localStorage`에 저장되어 새로고침해도 유지됩니다.

## 기술
- 의존성 없는 단일 정적 `index.html` (폰트만 외부)
- Vercel은 빌드 설정 없이 그대로 배포됩니다.
